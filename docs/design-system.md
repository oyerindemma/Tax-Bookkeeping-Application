# Design System

## Visual Direction
A modern civic-finance aesthetic: calm, trustworthy, and confident. Clean layouts with generous whitespace, soft gradients, and subtle texture to feel premium without feeling cold.

## Typography
- Display: Fraunces (headings, key numbers)
- Body: IBM Plex Sans (content, UI)
- Numerals: tabular figures for financial values

## Colour Palette (Proposed)
- Ink: #1F2937
- Slate: #334155
- Mist: #E2E8F0
- Cloud: #F8FAFC
- Emerald: #0F766E
- Amber: #B45309
- Coral: #E11D48

## Layout and Spacing
- Use a 4px spacing grid
- Primary content width: 1200px desktop, 720px tablet
- Mobile-first layouts with large touch targets

## Components
- Elevated cards with soft shadows
- Single primary action per screen
- Inline validation with calm, human language
- Clear data tables with sticky headers

## Motion
- Page load: subtle fade + slide (150-200ms)
- Staggered list reveals for dashboards
- Progress indicators for sync and calculations

## Accessibility
- WCAG 2.2 AA contrast compliance
- Keyboard navigation for web
- Screen-reader friendly labels
- Motion reduced for prefers-reduced-motion

## Content Style
- Plain English with UK spelling
- Explain tax obligations as steps, not rules
- Avoid fear-based language; emphasise clarity and confidence
